# 打刻メモ機能 — 確定仕様

> Q&A: [qa.md](./qa.md)

## 概要

打刻（出勤・退勤）時に任意のメモ（備考）を入力・保存できる機能。
保存したメモは本人の勤怠履歴・当日状況、および上長のチーム詳細画面で確認できる。

## データモデル

| フィールド | 型 | 制約 | 備考 |
|-----------|---|------|------|
| clock_in_memo | VARCHAR(100) | nullable | 出勤打刻時のメモ |
| clock_out_memo | VARCHAR(100) | nullable | 退勤打刻時のメモ |
| clock_in_memo_updated_at | TIMESTAMP WITH TIME ZONE | nullable | 出勤メモの最終編集日時 |
| clock_out_memo_updated_at | TIMESTAMP WITH TIME ZONE | nullable | 退勤メモの最終編集日時 |

- 1レコード（= 1出退勤ペア）に出勤用・退勤用の2つのメモを持つ
- 各メモは最大100文字
- 入力は任意（空欄でも打刻可能）

## 入力タイミング

- 出勤打刻時: 出勤メモを入力可能
- 退勤打刻時: 退勤メモを入力可能
- 打刻ボタンの近くに常時テキスト欄を表示する（モーダルは使わない）

## 編集

- 打刻後もメモを編集可能
- 編集可能期間: **当月中のみ**（翌月以降は読み取り専用）
- 編集UIの場所: **勤怠履歴画面**で該当行のメモをクリック（インライン編集）
- 編集すると `*_memo_updated_at` を更新し、画面に「(編集済み MM/DD HH:mm)」と表示
- 編集履歴（変更前の値）は保持しない

## 表示

### 本人向け

| 画面 | 表示内容 |
|------|---------|
| 打刻画面（本日の打刻状況） | 出勤メモ・退勤メモを表示 |
| 勤怠履歴画面（月次テーブル） | 出勤メモ・退勤メモを列追加。編集済みの場合は「(編集済み MM/DD HH:mm)」表示 |

### 上長向け

| 画面 | 表示内容 |
|------|---------|
| チーム勤怠詳細画面（新規） | チームメンバーの日次一覧＋メモを閲覧可能。既存の /team からメンバー行をクリックして遷移 |

- 上長は閲覧のみ（メモの編集権限は本人のみ）

## API 変更

| エンドポイント | 変更内容 |
|--------------|---------|
| POST `/api/attendance/clock-in` | リクエストボディに `memo` (optional, max 100) を追加 |
| POST `/api/attendance/clock-out` | リクエストボディに `memo` (optional, max 100) を追加 |
| PATCH `/api/attendance/{id}/memo` | 新規追加。メモ編集用。当月内のレコードのみ許可 |
| GET `/api/attendance/history` | レスポンスに `clockInMemo`, `clockOutMemo`, `clockInMemoUpdatedAt`, `clockOutMemoUpdatedAt` を追加 |
| GET `/api/attendance/today` | レスポンスに同上フィールドを追加 |
| GET `/api/attendance/team/{employeeId}/history` | 新規追加。上長がメンバーの月次日次一覧＋メモを取得 |

## 画面変更

1. **打刻画面** — 出勤/退勤ボタンの近くにテキスト入力欄を追加
2. **本日の打刻状況** — メモを表示
3. **勤怠履歴テーブル** — メモ列追加 + インライン編集
4. **チーム勤怠画面** — メンバー行クリックで詳細画面へ遷移するリンク追加
5. **チーム勤怠詳細画面（新規）** — /team/{employeeId} でメンバーの日次一覧＋メモ閲覧

## スコープ外

- メモの編集履歴の保持
- メモの検索・フィルタリング
- 上長によるメモ編集
